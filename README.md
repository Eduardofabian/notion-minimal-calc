# Notion Minimalist Calculator 🧮

A sleek, aesthetic, and fully functional calculator widget designed specifically to be embedded directly into Notion workspaces. Built with a focus on minimalism to match Notion's native dark mode interface seamlessly.

## ✨ Features
* **Native Feel:** Designed with matching colors, shadows, and typography to look exactly like a native Notion block.
* **Minimalist UI:** Clean, distraction-free interface inspired by minimalist productivity tools.
* **Responsive Layout:** Adapts smoothly to different embed block sizes within your Notion pages.
* **Core Operations:** Supports all basic arithmetic operations (addition, subtraction, multiplication, and division).

## 🛠️ Tech Stack
* **HTML5:** Semantic structure and layout.
* **CSS3:** Custom styling, CSS Grid for button organization, and interactive hover effects.
* **JavaScript:** Vanilla JS for the core calculator logic and state management.

## 🚀 How to Use in Notion
1. Copy the live project URL (hosted via GitHub Pages).
2. Go to your Notion page.
3. Type `/embed` and press Enter.
4. Paste the URL into the embed link field.
5. Resize the block to your preferred dimensions!
